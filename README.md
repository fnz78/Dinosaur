## 🌵 About The Game
A creative, fully playable pixel-art endless runner inspired by the classic offline browser game. Built entirely with HTML, CSS, and vanilla JavaScript!

## ✨ Features
- **Authentic Pixel Art:** Custom CSS-drawn dinosaur, cacti, and birds, complete with subtle running and tail-wag animations.
- **Dynamic Physics:** Responsive jumping with peak deceleration for that perfect "hang time".
- **Web Audio API:** Synthesized retro sound effects for jumping, crashing, and scoring milestones!
- **Particle Effects:** Impact dust bursts when colliding with obstacles.
- **Progressive Difficulty:** The game smoothly accelerates and dynamically increases obstacle density as your score climbs, providing a constantly evolving challenge.
- **Dynamic Environment:** The ground texture subtly shifts and animates to break up visual monotony.
- **Customizable Controls:** Drag-and-drop the 'JUMP' button on mobile devices to position it wherever it's most comfortable.
- **Audio Controls:** Mute/Unmute button to toggle all game sound effects.
- **Tutorial Overlay:** A brief controls guide is displayed on the first launch before the game starts.
- **Dark Mode:** Toggleable dark mode for late-night running.
- **Reset HI:** Clear your saved high score anytime with the Reset HI button.
- **Exit Menu:** A dedicated exit screen with animated pixel-art text that automatically pauses the game and lets you safely resume or exit forever.

## 🔒 Security & API Keys
This game runs entirely in the browser using standard Web APIs (like the Web Audio API) and does not require any external API keys or backend services. If you plan to extend this project with external services (e.g., global leaderboards or AI features), a `.env.example` file is included to help you securely manage your environment variables.

## 🎮 How to Play
- **Start/Jump:** Press `Spacebar` or tap the screen.
- **Restart:** Press `R` or click the restart icon after a game over.
- **Pause:** Click the pause button in the top right.
- **Dark Mode:** Click the moon/sun icon to toggle themes.
- **Reset HI:** Click the RESET HI button to clear your high score.
- **Exit:** Click the EXIT button to open the exit menu.

*Keep running, keep jumping, and aim for the high score!*
